# Amazon-scraper

# Qué Hace

Esta aplicación busca productos en Amazon usando una palabra clave, extrae el título, imagen, calificación y número de reseñas de la primera página, y los muestra.

# Tecnologías

* Backend: Node, JavaScript, Express, Axios, JSDOM
* Frontend: Vite, HTML, CSS, JavaScript

# Cómo Empezar

Necesitas:
* Node.js y npm instalados

Pasos:

1.  Backend:
    * Abre una terminal en la carpeta "Backend".
    * Instala dependencias: "npm install"
    * Inicia el servidor: "node index.js" (Debería correr en "http://localhost:3000")

2.  Frontend:
    * Abre otra terminal en la carpeta "Frontend".
    * Instala dependencias: "npm install"
    * Inicia el servidor de desarrollo: "npm run dev"

3.  Uso:
    * Abre la dirección que te da Vite en tu navegador.
    * Escribe una palabra clave y haz clic en "Buscar".
